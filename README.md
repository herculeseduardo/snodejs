Projeto SNODEJS
==========

Framework para node afim de padronizar um desenvolvimento
que possibilita a escalabilidade.

Baseado em modules Stateless.

Como usar ?
===========

```bash
npm install
```

```bash
npm start
```

Como acessar ?
==============

* [http://localhost:8080](http://localhost:8080)

Uso
===

* jade
* Bootstrap
* express
